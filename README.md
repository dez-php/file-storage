# file-storage
Simple File Storage based [dez-mvc-app](https://github.com/dez-php/dez-mvc-app)

### Upload
#### send *POST* request 
```
/upload
```

### For work with file

#### get direct link on file
```
/:hash/direct
```

#### deactivate file
```
/:hash/deactivate
```

#### activate file
```
/:hash/activate
```

#### real revoming file
```
/:hash/remove
```
